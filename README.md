# Personal-Mixed-Small-Projects-Portfolio
This repository contains some of my personal projects built using JavaFX, exploring a variety of creative approaches. Each project represents an opportunity for me to see how each one reflects creative experimentation and iterative learning.

# Greeting Card Design (CardDesignerUpdated.java)
With this program, you can design your own greeting card. You can edit the text/messages and change the background theme to customize the card.

# Hungry Coyote (CollectionGameUpdated.java) (requires SpriteUpdated.java to work)
Play as a hungry coyote, and eat all the chickens on the screen to win!

# Hunting Season (HuntingSeason.java)
Arm yourself as a hunter in this deer-filled target-shooter. Shoot the bucks, but don't shoot the does!

# Write Chinese Characters (WriteChineseChars.java)
Practice your mastery on writing Chinese characters using custom JavaFX brush and canvas implementations!

# Meditation Player (requires Sprite2.java to Work)
A custom-built meditation player with moving bubbles!

# Make your own Pizza
Create your own custom pizza with custom JavaFX brush and canvas implementations!
